Guía de ejecución para Pure Inka Foods

Para comenzar necesitas tener instalado Node.js en la versión 18 o superior y tener MySQL Server ejecutándose. También asegúrate de usar un navegador moderno como Chrome o Firefox.

Primero debes configurar la base de datos ejecutando los archivos SQL en orden. Empieza con el archivo DDL-3.sql para la estructura de la base de datos y luego ejecuta DML-v1-InkaFood.sql para cargar los datos del catálogo y las categorías.

Para iniciar el servidor backend ingresa a la carpeta Backend e instala las dependencias usando el comando npm install. Luego debes configurar el entorno modificando el archivo env colocando tu contraseña de base de datos en la variable DB_PASS y una clave secreta en JWT_SECRET. Finalmente inicia el servidor con el comando npm run dev.

Para iniciar la web frontend ingresa a la carpeta Frontend e instala las dependencias con el comando npm install. Después inicia la aplicación con npm start. Cuando termine de cargar podrás acceder ingresando a la ruta localhost en el puerto 4200 de tu navegador.

Si deseas validar que el sistema funciona correctamente puedes correr npm test dentro del backend y ng test en el frontend. Para revisar el estado del sistema visita la ruta api/health en el puerto 5000 de tu localhost.
